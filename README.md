# 🚀 Pagina Web Code-Digital

Pagina web donde se muestra informacion de la empresa Code Digital, sus servicios, proyectos, clientes y un formulario de contacto. Ademas de un Dashboard para la administracion de la pagina.

---

## 🧰 Tecnologías utilizadas

* PHP
* MySQL
* JavaScript
* HTML
* Tailwind CSS
* XAMPP

---

## ⚙️ Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/desarrollocodedigital/Code-Digital.git
```

2. Mueve el proyecto a la carpeta de XAMPP:

```bash
C:/xampp/htdocs/
```

3. Importa la base de datos:

* Abre `http://localhost/phpmyadmin`
* Crea una base de datos
* Importa el archivo `.sql`

4. Configura tu conexión a la base de datos en el archivo correspondiente:

- Host: localhost
- Usuario: tu_usuario
- Contraseña: tu_contraseña
- Base de datos: tu_base_de_datos

```

5. Ejecuta el proyecto:

```bash
http://localhost/Code-Digital
```
